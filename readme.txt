== Description ==

Need some inspiration? When activated Success Quotes plugin will show random success quotes in the top of your Wordpress admin page. This plugin is inspired from the famous 'hello dolly' plugin.


== Installation ==



1. Upload `success.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
